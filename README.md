# Aerospace Datalink Simulator
A project focused on learning aerospace datalink communications processes and systems. This project is meant to explore and learn about basic message passing to ACARS AND FANS/CPDLC communications.

# Project Goal
Develop a simulation that models how aircrafts exchange digital messages with ground-based stations.

Key Learning Points: (remove this and make each key learning point included in the dev phase?)
- Datalink architecture
- ACARS message routing
- FANS/CPDLC communications
- ADS-C reporting
- Standards implementation

## Phase 1 - Basic Message Passing Simulation (Current)
- Aircraft writes a message
- Message enters CMU
- Message reaches destination (determined by aircraft frequency)
- Reply is sent
- Aircraft receives reply
